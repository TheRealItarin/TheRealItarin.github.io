How was the website was created?

The website is using the built in webserver provided by github. I choose this solution, because I already have experience with it, it's easy to use, and even easier to maintain the webpage. Another strong reason to choose this is because it's completely free. For exact details about the website's creation, please refer to official documentation: https://docs.github.com/en/pages/getting-started-with-github-pages/creating-a-github-pages-site
